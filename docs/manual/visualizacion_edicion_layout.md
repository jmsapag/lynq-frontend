# Edición del Layout del Dashboard

LYNQ te permite personalizar completamente la disposición de los widgets en tu Dashboard Principal para que puedas enfocarte en las métricas más importantes para ti.

## Activar el Modo Edición

1.  Navega al Dashboard Principal.
2.  Busca el botón **"Editar Layout"** (o similar) en la parte superior derecha del área de contenido. 3. Haz clic en él. La interfaz cambiará ligeramente para indicar que estás en modo edición. Aparecerá una barra lateral con los widgets disponibles y las zonas del dashboard se marcarán como áreas donde puedes soltar widgets.

## Sistema Drag-and-Drop (Arrastrar y Soltar)

La personalización se realiza mediante un sistema intuitivo de arrastrar y soltar.

- **Añadir Widgets:**

  1.  En la **barra lateral de widgets**, busca el widget que deseas añadir (pueden estar categorizados por tipo: KPI, Gráfico, etc.).
  2.  Haz clic sostenido sobre el widget elegido.
  3.  Arrastra el widget hacia una de las **zonas de destino** (`Drop Zone`) resaltadas en el layout principal.
  4.  Suelta el widget en la zona deseada.

- **Mover Widgets:**

  1.  Haz clic sostenido sobre un widget que ya esté en el dashboard.
  2.  Arrastralo a una nueva zona de destino o a una posición diferente dentro de la misma zona.
  3.  Suelta el widget. Los demás widgets se reacomodarán si es necesario.

- **Eliminar Widgets:**
  1.  Pasa el cursor sobre el widget que deseas eliminar.
  2.  Debería aparecer un icono de "Eliminar" (una 'X' o un icono de papelera) en una esquina del widget.
  3.  Haz clic en ese icono. El widget será removido del layout (pero seguirá disponible en la barra lateral para añadirlo de nuevo si cambias de opinión).

## Selección de Layout Base (Opcional)

Es posible que tengas la opción de elegir entre diferentes estructuras base para tu dashboard (ej. una columna, dos columnas, layout mixto) antes de empezar a añadir widgets.

## Guardar Cambios

Una vez que estés satisfecho con la nueva disposición de tus widgets:

1.  Busca el botón **"Guardar Layout"** (o similar) que aparece mientras estás en modo edición. 2. Haz clic en él. Tus cambios se guardarán y la interfaz volverá al modo de visualización normal.

Tu layout personalizado se cargará cada vez que visites el dashboard.

# Comparativa de Datos

La sección de **Comparativa** te permite analizar y contrastar las métricas de afluencia entre diferentes ubicaciones o diferentes periodos de tiempo para una misma ubicación. Esto es muy útil para:

- Identificar qué locales tienen mejor rendimiento.
- Entender el impacto de campañas de marketing específicas comparando antes y después.
- Analizar patrones estacionales comparando el mismo periodo en años diferentes.
