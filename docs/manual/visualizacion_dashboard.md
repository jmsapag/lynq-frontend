# Dashboard Principal

El Dashboard Principal (también llamado Overview) es tu centro de comando para visualizar las métricas clave de afluencia y tráfico peatonal de tus ubicaciones.

## Filtros de Datos

En la parte superior del dashboard, encontrarás filtros esenciales para ajustar los datos que se muestran:

1.  **Selector de Fechas:**
    - Permite elegir un rango de fechas específico (Desde - Hasta).
    - Ofrece preselecciones comunes (Hoy, Ayer, Últimos 7 días, Últimos 30 días, Mes actual, Mes anterior, etc.).
    - - Los datos de todos los widgets se actualizarán para reflejar el periodo seleccionado.

2.  **Selector de Ubicaciones/Sensores:**
    - Permite seleccionar una o varias ubicaciones (locales) para visualizar sus datos agregados o individuales.
    - Si tienes acceso a múltiples locales, puedes elegir ver "Todos" o seleccionar específicos.
    - (Si aplica) Podría permitir seleccionar sensores individuales dentro de una ubicación. \* Haz clic en el botón de selección para abrir el modal y marcar las ubicaciones deseadas.

Una vez aplicados los filtros, los widgets del dashboard se actualizarán automáticamente.

## Área de Widgets

El cuerpo principal del dashboard está compuesto por **Widgets**. Cada widget muestra una métrica o gráfico específico. Consulta la sección [Widgets y Métricas](./visualizacion_widgets.md) para más detalles sobre cada uno.

El layout y los widgets mostrados por defecto pueden variar, pero puedes personalizar tu vista. Consulta [Edición del Layout](./visualizacion_edicion_layout.md) para aprender cómo.
