# Configuración de Reportes Semanales

LYNQ te permite configurar reportes automáticos que se envían por correo electrónico con un resumen de las métricas clave y los gráficos que elijas. Esto te ayuda a mantenerte informado sobre el rendimiento sin necesidad de entrar a la plataforma constantemente.

## Acceder a la Configuración de Reportes

1.  En la barra de navegación lateral, haz clic en **"Reportes"**.
2.  Verás una lista de los reportes que ya tienes configurados (si los hay) y un botón para crear uno nuevo.

## Crear o Editar un Reporte

1.  Haz clic en **"Crear Nuevo Reporte"** o en el icono de edición de un reporte existente.
2.  Se abrirá un modal o una página de configuración.
3.  **Configuración General:**

    - **Nombre del Reporte:** Dale un nombre descriptivo (ej. "Resumen Semanal - Tiendas Principales").
    - **Frecuencia:** Selecciona la frecuencia (actualmente parece ser semanal).
      - **Día y Hora de Envío:** Especifica qué día de la semana y a qué hora quieres recibir el reporte. Asegúrate de considerar la zona horaria.
    - **Destinatarios:** Ingresa las direcciones de correo electrónico de las personas que deben recibir el reporte. Pueden ser múltiples correos separados por coma.
    - **Ubicaciones Incluidas:** Selecciona las ubicaciones específicas cuyos datos quieres incluir en este reporte.
    - **Activo/Inactivo:** Un interruptor para habilitar o deshabilitar el envío automático de este reporte.

4.  **Configuración del Contenido:**

    - Selecciona un **Layout Base** para el reporte (ver [Layouts de Reporte](./reportes_layout.md)).
    - Personaliza los **Widgets** que se incluirán en el reporte usando el sistema drag-and-drop (ver [Layouts de Reporte](./reportes_layout.md)).

5.  **Guardar:** Haz clic en "Guardar" para almacenar la configuración del reporte.

## Gestionar Reportes Existentes

En la página principal de "Reportes", puedes:

- Ver la lista de tus reportes configurados.
- Activar o desactivar reportes rápidamente.
- Editar la configuración de un reporte.
- Eliminar un reporte que ya no necesites.
