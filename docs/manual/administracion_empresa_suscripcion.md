# (ADMIN) Gestión de Suscripción

Como **Administrador (ADMIN)**, además de [visualizar el estado de la suscripción](../04_Gestion_Cuenta/02_Suscripciones.md), tienes acceso a funcionalidades para gestionar activamente el plan y los pagos de tu empresa. Las opciones exactas dependerán de si tu suscripción se gestiona automáticamente vía **Stripe** o de forma **Manual**.

## Acceder a la Gestión de Suscripción

1.  Navega a la sección **"Facturación"** o **"Suscripción"** desde la barra lateral.

## Suscripción vía Stripe (Automática)

Si tu empresa se suscribió a través de la integración con Stripe, podrás realizar las siguientes acciones:

- **Gestionar Métodos de Pago:**
  - Ver los métodos de pago guardados (tarjetas).
  - Añadir un nuevo método de pago haciendo clic en **"Añadir Método de Pago"**. Serás redirigido a una interfaz segura de Stripe para ingresar los datos.
  - Eliminar métodos de pago existentes.
  - Establecer un método de pago como predeterminado para los cobros automáticos.
  - **Ver Historial de Facturas:** Acceder a las facturas generadas por Stripe.
  - **Cambiar Plan / Modificar Suscripción:**
  - Si existen diferentes planes o niveles, podría haber una opción para **"Cambiar Plan"**. \* Si el precio se basa en la cantidad de sensores, la actualización podría ser automática al añadir/eliminar ubicaciones/dispositivos.
    - **Cancelar Suscripción:**
  - Puede haber un botón para iniciar el proceso de cancelación.
  - ## Suscripción Manual

Si tu suscripción se gestiona manualmente mediante acuerdo con el equipo de LYNQ, el proceso es diferente:

- **Visualización de Estado:** Verás claramente si la suscripción está activa o si está **"Pendiente de Pago"**.
- **Renovación / Subida de Comprobante:**
  1.  Cuando se acerque la fecha de vencimiento o si ya está vencida, deberás realizar el pago según lo acordado (transferencia, depósito).
  2.  Una vez realizado el pago, accede a la sección de Suscripción en LYNQ.
  3.  Busca la opción **"Subir Comprobante de Pago"**.
  4.  Adjunta el archivo digital de tu comprobante (PDF, imagen).
  5.  Haz clic en **"Enviar"**.
- **Activación:** El equipo de LYNQ recibirá una notificación, verificará el pago y actualizará el estado de tu suscripción manualmente en el sistema. Recibirás una notificación o verás el cambio de estado en la plataforma una vez aprobado.
  Si tienes dudas sobre tu tipo de suscripción o el proceso de pago, contacta al [Soporte](../06_Soporte/01_Crear_Ticket.md) o a tu ejecutivo de cuenta LYNQ.
