# Inicio de Sesión

Acceder a la plataforma LYNQ es sencillo.

## Pasos para Iniciar Sesión

1.  Abre tu navegador web y ve a la URL de LYNQ proporcionada por tu administrador. 2. Verás la pantalla de inicio de sesión.
2.  Introduce tu **dirección de correo electrónico** registrada.
3.  Introduce tu **contraseña**.
4.  Haz clic en el botón **"Iniciar Sesión"**.

Si las credenciales son correctas, serás redirigido al Dashboard Principal de LYNQ.

## ¿Olvidaste tu contraseña?

Si no recuerdas tu contraseña, puedes restablecerla:

1.  En la pantalla de inicio de sesión, haz clic en el enlace **"¿Olvidaste tu contraseña?"**.
2.  Serás redirigido a una página donde deberás ingresar tu **dirección de correo electrónico** registrada.
3.  Haz clic en el botón para solicitar el restablecimiento.
4.  Recibirás un correo electrónico con instrucciones y un enlace para crear una nueva contraseña. Sigue las instrucciones del correo.

# Interfaz General

Una vez que inicias sesión, te encontrarás con la interfaz principal de LYNQ. Está diseñada para ser intuitiva y permitirte acceder rápidamente a la información que necesitas.

Los elementos principales de la interfaz son:

## 1. Barra de Navegación Lateral (Sidebar)

Ubicada en el lado izquierdo, esta barra contiene los enlaces a las diferentes secciones de la plataforma. Las opciones visibles pueden variar según tu rol (si eres administrador o un usuario estandar).

Las secciones comunes incluyen:

- **Dashboard/Overview:** La vista principal con tus métricas clave.
- **Comparación:** Herramienta para comparar datos entre ubicaciones o periodos.
- **Reportes:** Configuración de reportes semanales automáticos.
- **Ubicaciones:** (Visible para ADMIN y STANDARD con acceso) Lista de locales.
- **Gestionar Usuarios:** (Solo ADMIN) Administración de usuarios y permisos.
- **Facturación/Suscripción:** Gestión de tu plan y pagos.
- **Ayuda/Soporte:** Acceso a la creación de tickets.
- **Mi Perfil:** Configuración de tu cuenta personal.

## 2. Rapido acceso

En la parte superior de la pantalla, encontrarás:

- **Título de la Página:** Indica en qué sección te encuentras.
- **Acciones Rápidas:** Botones relevantes para la página actual (ej. "Editar Layout", "Exportar").

Y en la esquina inferior izquierda:

- **Icono de Usuario/Perfil:** Acceso rápido a tu perfil y opción para cerrar sesión.

## 3. Área de Contenido Principal

Es el espacio central donde se muestra la información principal de cada sección (gráficos, tablas, formularios, etc.).

## Tour Guiado

La primera vez que inicies sesión, se activara **Tour Guiado** para presentarte las funcionalidades básicas. Te recomendamos seguirlo para familiarizarte rápidamente con la plataforma.
